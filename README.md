# Interactive-CV-Stephany-Gonzalez

# Hello and welcome!

# This Interactive report was designed to represent key areas of my professional background:

- Profile - basic personal info and blog link
- Work History - company, duration, location, and responsibilities
- Education - degrees, schools, and duration
- Projects - Skills and duration
- Skills - skill name, category, proficiency, and duration
- Certifications - credentials, issuers, and badge URLs
- Work Skills - bridge table connecting Work History and Skills
- Date - used to enable timelines and calculate durations

## Clicking on each of the icons will allow you to see more detailed information about each data category, as well as links to LinkedIn, GitHub, and others.
